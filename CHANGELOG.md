# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-27

### Added
- Clean, minimalist dark theme design
- Topic-based blog organization (Talent, Product, Writing, Systems, Crypto)
- Individual post pages with reading time estimates
- Topic filtering pages
- RSS feed generation
- Comprehensive TypeScript types
- Centralized configuration in `config.ts`
- JSDoc documentation for all utility functions
- Mobile-responsive design
- Accessibility improvements

### Changed
- Complete redesign from original template
- Refactored all components for better maintainability
- Improved folder structure and organization
- Enhanced CSS with better commenting

### Removed
- Old placeholder content
- Unused image assets
- Deprecated Atkinson fonts
